======================================= Инструкция ==============================================
1.Установить docker.io и docker-compose на хост где будет осуществляться сборка.

2.После скачивания архива в этой же папке нужно выполнить команду для сборки и запуска контейнера с apache
bigtop в docker:
docker-compose up -d --build

3.После сборки проваливаемся в запущенный контейнер с bigtop:
docker exec -it e365ec55550b  /bin/bash - P.S id контейнера указываем свой этот как пример.

4. Пользуемся.
